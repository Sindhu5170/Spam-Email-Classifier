# 📧 Spam Email Classifier

## Overview

Spam Email Classifier is a Machine Learning project that detects whether an email is Spam or Ham (Not Spam) using Natural Language Processing and the Naive Bayes algorithm.

## Features

- Spam Detection
- Text Classification
- Machine Learning Based
- Beginner Friendly

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- CountVectorizer
- Naive Bayes

## How to Run

1. Install requirements

pip install -r requirements.txt

2. Run project

python spam_classifier.py
